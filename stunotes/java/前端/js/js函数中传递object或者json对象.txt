怎么在onclick事件中传递对象参数，这个问题困扰了好久，终于测试通过，遂记录之。

var user = {id:1, name:'zs', age:20};

var ele = '<a onclick="edit(' + JSON.stringify(user).replace(/"/g, '&quot;') + ');">修改</a>';

或者 

var ele = '<a onclick="edit(\'' + JSON.stringify(user).replace(/"/g, '&quot;') + '\');">修改</a>';

前者取到的是json对象，后者取到的是json字符串。
